## README.md 작성할때 참고 (기호)

### 제목을 입력할 때 사용되며 #의 개수가 많아질수록 크기가 줄어듭니다.

### Ctrl + Shift + V : 미리보기

1.  '>' 인용문을 작성할 때 이용합니다. 개수가 늘어날 경우 중첩으로 적용됩니다.
2.  '-' 순서없는 목록을 작성할 때 사용됩니다.
3.  1. 2. 3. 순서있는 목록을 작성할 때 사용됩니다.
4.  `___` 인라인 코드 목록을 작성 시 사용됩니다.
5.  ' ``` ' 코드 내용을 작성 시 사용됩니다.
6.  | \_\_ | 표를 작성할 때 사용됩니다.
